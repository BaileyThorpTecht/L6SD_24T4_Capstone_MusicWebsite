# L6SD_24T4_Capstone_MusicWebsite

## Create Virtual Enviroment After Cloning !!!
- In VSCode, press Ctrl + P to open the top bar.
- Type ">Python: Create Enviroment..."
- Select venv
- Select your python installation

Pip install packages
- django
- django-bootstrap5
- pillow
- django-crispy-forms
- crispy_bootstrap5

Extentions Needed:
- SQLite
- SQLTools SQLite
- SQLTools
